기술 문서, 코드 주석, 보고서, 위키 등 작업지시자 대상 작성물에서 **"산수"** 표현은 사용하지 않는다. **"계산"** 으로 표현한다.

**Why:** "산수" 는 너무 원초적인 단어로 기술 문서 / 보고서 톤에 부적합. "계산" 이 맥락 정합.

**How to apply:**
- 코드 주석, troubleshooting 문서, 위키, 결과보고서, 단계 보고서, 메모리 등 모든 작성물에 적용
- 검토 단계: 문서 작성 후 grep "산수" 자동 점검
- 같은 의미의 표현 (수식, 공식, 식, 산식) 중 정합한 것 선택
