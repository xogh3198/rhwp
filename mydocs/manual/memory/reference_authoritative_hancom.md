**작업지시자 정답지 환경:**
- 한컴 2010 (한컴오피스 2010, 한글)
- 한컴 2022 (한컴오피스 2022, 한글)

작업지시자가 페이지 분할 / 페이지네이션 / 시각 결과 검증 시 위 두 환경의 한컴 편집기 출력을 권위 기준으로 사용. PR #360 정황 (2026-04-27) 에서 확인됨.

**정답지 아닌 경우:**
- 한컴 Viewer (한글뷰어, 편집기 아님)
- 한컴 외부 변환 도구 (hwp2pdf 등)
- macOS 인쇄 다이얼로그를 통한 PDF 출력 (한컴 외부 경로)
- 다른 한컴 버전 (한컴 2018 등) 의 결과는 미확인

**적용:**
1. PR 검토 시 "PDF 와 일치" 검증 결과가 한컴 2010 또는 한컴 2022 가 아니면 작업지시자 환경 재검증 필수
2. 컨트리뷰터의 PDF 출력 방법 확인:
   - 한컴 편집기 (2010, 2022) → 작업지시자 정답지와 비교 가능성 높음
   - 한컴 Viewer / 외부 도구 → 작업지시자 정답지와 다를 가능성 → 재검증 필수
3. 작업지시자 환경의 정확한 버전 / OS 등은 향후 task 에서 commit 메시지 / 보고서에 명시 권장
