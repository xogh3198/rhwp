# 수행계획서

## 1. 배경

Issue #265에서 HWP 3.0 파일 감지 + 친절한 에러 메시지를 구현 완료(2026-04-24). 이후 `src/parser/hwp3/` 디렉토리에 HWP 3.0 전체 파서 구현이 진행됐으나 git untracked 상태로 존재. CLAUDE.md 하이퍼-워터폴 프로세스(이슈→브랜치→계획서→구현)를 거치지 않고 구현이 진행된 상태를 정비하고 미완성 항목을 완성한다.

## 2. 목표

- `mydocs/tech/한글문서파일구조3.0.md` 스펙 기준으로 HWP 3.0 파서 구현
- 변환 결과(`Document` IR)를 기존 WASM API로 전달하여 rhwp-studio HTML 에디터에서 렌더링
- 전달 경로: `parse_hwp3()` → `Document` IR → `parse_document()` 라우팅 → `HwpDocument::new()` → `render_page_svg/html()`

- 표(Table), 그림(Picture), 그리기 객체 기본 파싱
- 머리말/꼬리말, 각주/미주 기본 구조
- `parse_document()` → HWP3 라우팅 연결
