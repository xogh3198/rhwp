# Task #429 수행 계획서 — 표 셀 배경 image fill 미구현

## 배경

[이슈 #429](https://github.com/edwardkim/rhwp/issues/429) — `samples/aift.hwp` 41쪽 표 셀 안 이미지가 SVG 출력에 누락.

작업지시자 통찰 (결정적):
- **셀의 배경 그림** (BorderFill 의 image fill mode) — rhwp 미구현
- 이미지가 셀 안 paragraph control 이 아닌 **BorderFill 의 fill type=Image**

## 결함 본질

### 정황

`samples/aift.hwp` 페이지 41 의 표 pi=496 (2x1, RowBreak, TAC):
- 셀[0] r=0, c=0, **bf=175** — `border_fill[175] fill_type=Image image(bin_id=14, mode=None)`
- 셀[0] paragraph p[0] **ctrls=0** (셀 안 control 없음)
- 셀[1] r=1, c=0, bf=1 (Solid white) — 캡션 텍스트 ("[그림 8] AI 팩토리 정책 개념도 및 참여기업")

→ 그림은 **셀 BorderFill 의 image fill** 로 저장되어 있지만 rhwp 가 렌더링 안 함.

## 정정 방향

`render_cell_background` 에 image_fill 분기 추가 — zone 처리 (table_layout.rs:303-316) 와 동일 패턴:

```rust
// 정정 후
pub(crate) fn render_cell_background(
    &self,
    tree: &mut PageRenderTree,
    cell_node: &mut RenderNode,
    border_style: Option<&ResolvedBorderStyle>,
    cell_x: f64, cell_y: f64, cell_w: f64, cell_h: f64,
    bin_data_content: &[BinDataContent],  // 추가
) {
    // ... 기존 fill_color / pattern / gradient 처리
    
    // [Task #429] image fill 처리
    if let Some(ref img_fill) = border_style.and_then(|bs| bs.image_fill.as_ref()) {
        if let Some(img_content) = find_bin_data(bin_data_content, img_fill.bin_data_id) {
            let img_id = tree.next_id();
            let img_node = RenderNode::new(
                img_id,
                RenderNodeType::Image(ImageNode {
                    fill_mode: Some(img_fill.fill_mode),
                    ..ImageNode::new(img_fill.bin_data_id, Some(img_content.data.clone()))
                }),
                BoundingBox::new(cell_x, cell_y, cell_w, cell_h),
            );
            cell_node.children.push(img_node);
        }
    }
}
```

호출부 (table_layout.rs:264, 1154, table_partial.rs:240) 모두 `bin_data_content` 인자 전달 정합 필요.

- zone 배경 image fill (이미 구현) 영역 회귀 없음 점검
