## 본질 — RowBreak 표 정책 vs rowspan 보호 블록 정책 모순

| 요소 | 정합 |
|------|------|
| 표 정책 (HWP `TablePageBreak::RowBreak`) | **행 경계에서 나눔** — 인트라-로우 분할 금지, 행 단위 분할 허용 |
| Task #398 의 `BLOCK_UNIT_MAX_ROWS=3` 정책 | rowspan>1 셀 묶음 보호 → row 단위 분할 차단 |
| HWP 의도 | 셀[8] (rs=2) 묶음 + 표 정책 RowBreak → **행 단위 분할 허용** |

## 정정 방향

`snap_to_block_boundary` (+ 호출부의 보호 블록 가드) 에서 표 정책 검사 — **RowBreak 표는 보호 블록 정책 비적용**.
