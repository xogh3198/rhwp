## 진단 본질

작업지시자 통찰 정합: 회귀 본질이 \`map_pua_bullet_char\` 의 일반 적용이 아니라, **매핑 표의 범위 한계** — Basic PUA (0xF020~0xF0FF) 만 커버, **Supplementary PUA-A (0xF0000+) 및 일부 Basic PUA 영역 (예: U+0F53A) 미커버**.

## 회귀 본질 정합

| 영역 | 정합 |
|------|------|
| Basic PUA 매핑 표 적용 영역 (예 U+0F0F2 ⇩ → ◆ 정정 필요) | 매핑 표의 의도가 정확하지 않은 케이스 |
| Basic PUA 매핑 표 미정의 영역 (예 U+0F0E8, U+0F53A) | 매핑 표 범위 확장 필요 |
| Supplementary PUA-A (0xF0000+) | 매핑 표 자체 본 영역 미지원 — Supplementary 영역 추가 필요 |
